# Rock_Paper_Scissors-C_Programming
This program is a simple Rock-Paper-Scissors game developed in the C programming language where a user plays against the computer. The program continuously runs using an infinite while loop and asks the user to enter a move: 0 for Rock, 1 for Paper, 2 for Scissor, and 9 to exit the game. The computer generates its move randomly using the rand() function. After taking both moves, the program compares them using multiple if-else conditions to determine the winner and displays whether the user won, the computer won, or the game ended in a draw. The program also uses important programming concepts such as loops, conditional statements, random number generation, user input/output functions, and the exit() function to terminate the game when the user chooses to quit.
